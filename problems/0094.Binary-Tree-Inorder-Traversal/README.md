# 0094. [Binary Tree Inorder Traversal](https://leetcode.com/problems/binary-tree-inorder-traversal/)

## 題目

Given a binary tree, return the inorder traversal of its nodes’ values.

**Example:**

```
Input: [1,null,2,3]
   1
    \
     2
    /
   3

Output: [1,3,2]
```

## 題目大意

用 inorder 的方式走遍一個二元樹。

## 思路

依照題意，使用遞迴的方式實作。
